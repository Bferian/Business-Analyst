# 📊 GlobalCorp BI Dashboard — Business Intelligence Project

> Interactive executive dashboard for global business performance analysis, developed as a portfolio project for the **Business Analyst** role.
> Fictional data · 750 transactions · 20 countries · 6 regions · 2020–2024

---

## 🔍 About the Project

This project simulates the real work of a Business Analyst at a **multinational company**, answering the question:

> *"How is our global performance and which regions/categories deserve strategic attention?"*

The dashboard was built entirely in **HTML/CSS/JavaScript**, with no frameworks, and can be opened directly in the browser — zero installation required.

## 💡 Key Insights

| # | Insight | Impact |
|---|---------|--------|
| 1 | **North America represents 35% of total revenue** | High |
| 2 | **Software has a 69% margin** — the most profitable category | High |
| 3 | **Online channel has the best margin** due to no physical structure costs | Medium |
| 4 | **Africa has only 8% of revenue** but 59.9% margin — high untapped potential | High |
| 5 | **Average YoY growth of +8%** from 2020 to 2024 — consistent trend | Medium |
| 6 | **Revenue peaks in Q4** (Oct–Dec) across all regions | Medium |

---

## 🎯 Strategic Recommendations

1. **Accelerate SaaS B2B in Europe and North America** — highest margin (69%) + premium markets
2. **Enter the African market** with lower-ticket products (Food & Fashion)
3. **Migrate distributors to the Online channel** — better margin, lower operational cost
4. **Prepare Q4 in advance** — increase stock 60 days before for Electronics and Fashion

## ⚙️ How to Run

No installation needed. Download the file and double-click to open in Chrome, Firefox or Edge.

Requires internet connection to load charts (Chart.js and Leaflet via CDN).

---

## 🧰 Technologies Used

| Technology | Usage |
|------------|-------|
| **HTML5 / CSS3** | Dashboard structure and layout |
| **JavaScript (Vanilla)** | Filters, navigation and chart logic |
| **Chart.js 4.4** | Line, bar, pie, scatter and donut charts |
| **Leaflet.js 1.9** | Interactive world map with bubble chart |
| **Python (pandas)** | Fictional data generation and aggregation |
